# Прогнозирование сердечных заболеваний

Это приложение на Streamlit, которое использует модель машинного обучения для прогнозирования наличия сердечных заболеваний на основе введенных пользователем данных.

## Установка зависимостей
Убедитесь, что у вас установлены необходимые библиотеки:

```bash
pip install streamlit pandas
```

## Запуск приложения
Запустите приложение с помощью команды:

```bash
streamlit run app.py
```

## Использование приложения
1. Откройте веб-браузер и перейдите по адресу, указанному в выводе команды.
2. Введите данные пациента в боковой панели.
3. Нажмите кнопку "Предсказать", чтобы получить прогноз от модели.
4. Просмотрите предсказание и, при необходимости, внесите изменения в введенные данные.

## Описание полей данных
- **Возраст (Age):** Возраст пациента в годах.
- **Пол (Sex):** Пол пациента (М - мужской, Ж - женский).
- **Тип боли в груди (ChestPainType):** Описание типа боли в груди (TA: Типичная стенокардия, ATA: Атипичная стенокардия, NAP: Неангинальная боль в груди, ASY: Асимптоматическая).
- **Покойное давление (RestingBP):** Покойное артериальное давление в мм рт. ст.
- **Холестерин (Cholesterol):** Уровень холестерина в сыворотке в мм/дл.
- **Уровень сахара натощак (FastingBS):** Уровень сахара натощак (1: Если уровень сахара > 120 мг/дл, 0: В противном случае).
- **Электрокардиограмма в покое (RestingECG):** Результаты электрокардиограммы в покое (Normal: Нормально, ST: Аномалии в ST-T волне, LVH: Утолщение левого желудочка).
- **Максимальная частота сердечных сокращений (MaxHR):** Максимальная частота сердечных сокращений.
- **Стенокардия при физической нагрузке (ExerciseAngina):** Наличие стенокардии при физической нагрузке (Y: Да, N: Нет).
- **Старое углубление ST (Oldpeak):** Старое углубление ST (измеряется в миллиметрах).
- **Наклон пика упражнения ST (ST_Slope):** Наклон пика упражнения ST (Up: Восходящий, Flat: Плоский, Down: Нисходящий).
- **Прогноз (HeartDisease):** Прогноз наличия сердечного заболевания (1: Сердечное заболевание, 0: Нормально).

Модель и предобработчик сохраняются и загружаются с использованием библиотеки pickle. Для сохранения модификаций ваших данных, вы можете добавить соответствующий функционал в код приложения.
